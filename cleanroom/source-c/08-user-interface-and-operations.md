# Source C — user interface and operations

## Dashboard

Entry dashboard summarizes IP usage and alerts depending on configuration.

## Tools surface

Extensive **`app/tools`** subtree: racks, routing, DHCP, PowerDNS, search, devices, locations, NAT, multicast, PSTN, vaults, favourites, documentation browser—**operator-centric** UX.

## Daily operations

Password change, temp shares, instructions, and logging support routine admin tasks.

## Hosting

Typical stack: **Apache or Nginx**, **PHP-FPM**, **MySQL/MariaDB**. Fits low-cost VMs and containers.
