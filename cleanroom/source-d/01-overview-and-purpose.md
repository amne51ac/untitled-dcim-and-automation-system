# Source D — overview and purpose

## Mission

Source D is a **full-featured asset management, DCIM, and CMDB** system for **data centers and back offices**, with **lifecycle workflows** and **data center visualization**. **Apache 2.0** per project metadata.

## Governance

Public README describes **“sources only”** maintenance: continued publishing without guaranteed community PR review; commercial support via separate channels—plan internal capacity if depending on timely upstream fixes.

## Audience

Organizations needing **procurement-to-retirement** rigor, **financial/vendor** metadata, and **facility visualization** alongside inventory—not only L3 network intent.
