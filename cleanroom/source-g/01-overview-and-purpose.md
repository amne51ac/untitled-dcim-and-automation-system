# Source G — overview and purpose

## Mission

Source G manages **data center infrastructure**—racks, power, cabling, and related reporting—for facilities of varying sizes. **GPLv3** per project README; originated in academic IT and continued as an open project.

## Scope

**Raised-floor** and **facility** orientation; not a full L3–L7 network service modeling platform like Source A.

## Development warning

README notes **GitHub as development**—production deployments should track **released** artifacts and community guidance.
